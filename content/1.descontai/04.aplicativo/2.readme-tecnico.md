# Descontai — APP

Aplicativo mobile desenvolvido pela **Koomit Software** para a plataforma Descontai — sistema de descontos, cupons e benefícios para usuários em estabelecimentos parceiros.

---

## Stack

| Camada | Tecnologia |
|---|---|
| Framework | Flutter (Dart `^3.9.2`) |
| State Management | Provider + Riverpod |
| Roteamento | GoRouter |
| HTTP | Dio |
| Backend | REST API (DigitalOcean) |
| Notificações | Firebase Cloud Messaging |
| Analytics / Crash | Firebase Analytics + Crashlytics |
| Auth local | Biometria (`local_auth`) + Secure Storage |
| QR | `qr_flutter` + `mobile_scanner` |

---

## Requisitos

- Flutter `>=3.29.0` (canal stable)
- Dart `^3.9.2`
- Android Studio / Xcode para builds nativas
- Arquivo `android/app/google-services.json` válido (Firebase Android)
- Arquivo `ios/Runner/GoogleService-Info.plist` válido (Firebase iOS)

---

## Configuração local

```bash
# Instalar dependências
flutter pub get

# Gerar código (Freezed / JSON serializable)
dart run build_runner build --delete-conflicting-outputs

# Rodar em modo debug
flutter run

# Rodar com URL de API customizada
flutter run --dart-define=BASE_URL=https://sua-api.exemplo.com
```

## Estrutura do projeto

```
lib/
├── core/
│   ├── config/        # Endpoints e cache de sessão
│   ├── controller/    # AppController (estado global)
│   ├── data/
│   │   ├── models/    # Modelos de resposta da API
│   │   └── services/  # Camada de serviços HTTP
│   ├── network/       # Cliente Dio
│   ├── router/        # Rotas GoRouter
│   └── theme/         # Tema Material 3
├── features/
│   ├── auth/          # Login, cadastro, reset de senha
│   ├── home/          # Feed principal
│   ├── search/        # Busca de estabelecimentos
│   ├── product_detail/# Detalhe do produto/cupom
│   ├── historico/     # Histórico de economias
│   ├── profile/       # Perfil e conta do usuário
│   └── support/       # Suporte / ajuda
└── shared/
    ├── theme/         # Paleta de cores
    ├── utils/         # Utilitários compartilhados
    └── widgets/       # Componentes reutilizáveis
```

## Build de produção

```bash
# Android APK
flutter build apk --release

# Android App Bundle (recomendado para Play Store)
flutter build appbundle --release

# iOS
flutter build ios --release
```

---

## Firebase

O projeto usa Firebase para push notifications, analytics e Crashlytics.  
Os arquivos de configuração (`google-services.json` e `GoogleService-Info.plist`) **não devem ser commitados** em repositórios públicos — utilize secrets do CI/CD para injetá-los no build.
