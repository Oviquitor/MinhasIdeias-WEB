O conteúdo é organizado em **quatro trilhas** encadeadas por senioridade. Os módulos
seguem uma **numeração linear global (01 → 53)** que atravessa as trilhas — ou seja, o
módulo 11 continua de onde o 10 parou, e assim por diante. Ao lado das trilhas há **9
projetos práticos**, um por nível, que transformam a teoria em sistema construído.

::callout{type="tip" title="Estude na ordem"}
As trilhas foram desenhadas para serem percorridas de cima para baixo. Cada módulo assume
o que veio antes; cada trilha prepara a próxima. Pular etapas deixa buracos que aparecem
lá na frente.
::

# 5. Projetos práticos

Cada nível de senioridade tem um **projeto** que reconstrói um sistema real, com aulas
navegáveis fase a fase (introdução → fases guiadas) e **código-companheiro** de referência
em Spring Boot 3 (Java 21) e ASP.NET Core 8.

| # | Projeto | Nível | Tema |
| --- | --- | --- | --- |
| 01 | Task Manager | Júnior | Primeiro CRUD completo com autenticação. |
| 02 | Blog Platform | Pleno | Value Objects tipados, casos de uso, testes. |
| 03 | E-commerce em microsserviços | Sênior | Gateway + serviços + RabbitMQ + resiliência (Polly/Resilience4j). |
| 04 | E-commerce com DDD | Arq. Software | Bounded contexts, event sourcing, SAGA. |
| 05 | FinanceOS | Arq. Soluções | Copiloto financeiro — multi-tenant, RNFs. |
| 06 | Vitrine SaaS | Arq. Soluções | SaaS multi-tenant com RLS e isolamento por tenant. |
| 07 | Legado e migração | — | Strangler Fig: monólito Node estrangulado por serviço novo dual. |
| 08 | Transformação digital enterprise | — | Projeto de processo (kit de templates), não de código. |
| 09 | AI-powered support platform | — | RAG + agente ReAct com cliente LLM provider-agnóstico. |

::callout{type="note" title="Módulos e projetos se amarram"}
Os projetos não são exercícios soltos: cada um consolida os módulos da sua trilha. A
última aula de cada módulo sugere o projeto correspondente, e o System Design do módulo 49
aparece de forma interativa no Playground. O objetivo é fechar o **loop teoria → prática →
teoria** — ver a próxima página, **Features e experiência**.
::

::callout{type="info" title="Sobre os números"}
São **53 módulos** numerados de 01 a 53 (alguns desdobrados em sub-módulos "a", como 19a e
24a, para separar assuntos que mereciam capítulo próprio sem quebrar a numeração), mais de
**600 aulas** de trilha e **168 aulas** distribuídas nos 9 projetos — todas escritas em
Java e C#.
::
