O RoadMap Academy não é um site de páginas estáticas com texto corrido. Cada recurso existe
para **fixar** o aprendizado e fechar o loop entre teoria e prática.

# 1. Aulas ricas

## 1.1 Abas Java | C#

Blocos de código que aparecem em Java seguidos de C# são agrupados automaticamente em
**abas** (Java é o padrão). O mecanismo é CSS puro — funciona sem JavaScript, direto no
HTML renderizado do Markdown. O aluno alterna entre as linguagens sem sair da aula.

## 1.2 Diagramas Mermaid renderizados

Todo bloco `mermaid` é renderizado como **diagrama visual** (não como código), client-side,
com tema escuro. Fluxos, arquiteturas e máquinas de estado aparecem desenhados dentro da aula.

## 1.3 Ganchos de prática

- **"Aplique agora no Playground"** — aparece quando a aula tem um exercício de System
  Design ligado a ela.
- **Chips "coloque em prática"** — na última aula de cada módulo, com os projetos da trilha.

# 2. Playground de System Design

Um canvas de **arrastar-e-soltar** (Vue Flow) para desenhar arquiteturas. O aluno monta a
solução com componentes de infraestrutura de uma paleta (11 categorias, 32 componentes:
load balancers, filas, caches, bancos, CDNs, etc.), conecta os nós e compara com uma
**solução de referência**.

::steps
### Modo livre
Uma tela em branco para desenhar qualquer arquitetura. Salvar, exportar e importar o
diagrama como JSON (persistido em `localStorage`).

### Modo exercício
Cenários guiados — encurtador de URL, rate limiter, sistema de notificações — cada um com
enunciado, checklist de requisitos e solução de referência, ligados às aulas do módulo de
System Design.
::

# 3. Ferramentas de estudo

::steps
### Busca global (Ctrl+K)
Uma paleta de busca sobre um índice de centenas de entradas — encontra aulas, módulos e
projetos por título e descrição, de qualquer lugar do site.

### Banco de entrevistas
Mais de mil perguntas extraídas das seções "Como aparece em entrevista" das aulas. Tem modo
**Explorar** (resposta inline) e modo **Flashcard** (baralho embaralhado, revelar,
autoavaliar), com progresso salvo e opção de revisar só as marcadas.

### Glossário
Centenas de termos técnicos organizados por categoria — de conceitos de sistemas
distribuídos a SRE, IA, segurança e negócio.

### Quizzes
Questionários por módulo para checar a retenção do conteúdo.
::

# 4. Progresso e conta

- **Login e cadastro** com autenticação real (Supabase Auth).
- **Progresso por aula** persistido: quando logado, salva no Supabase; anônimo, salva em
  `localStorage` e faz merge no login.
- **Dashboard**, **perfil**, **ranking** e **certificados** completam a experiência do aluno.

::callout{type="note" title="Escopo atual"}
Autenticação e progresso de aula estão persistidos de verdade no Supabase. **Ranking** e
**certificados** ainda são telas de demonstração (não persistem) — evolução planejada, não
um bug. Ver a **Arquitetura técnica** para os detalhes.
::

# 5. Descoberta

- **SEO** completo (Open Graph, Twitter cards, canonical, JSON-LD de Course na home,
  `useSeoMeta` por aula) e **sitemap** com centenas de URLs — todo o conteúdo público é
  indexável, o que faz sentido para uma plataforma de conhecimento.
