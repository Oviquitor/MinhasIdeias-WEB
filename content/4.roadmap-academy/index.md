**RoadMap Academy** é uma plataforma de estudo que organiza a carreira de um
engenheiro de software backend em **trilhas progressivas** — do **Júnior** ao
**Arquiteto de Soluções** — com aulas densas, projetos práticos que espelham
sistemas reais e ferramentas de fixação (playground de System Design, banco de
entrevistas, glossário e quizzes).

Não é um curso genérico: cada conceito é ensinado **em Java e em C#** lado a lado
(Spring Boot + ASP.NET Core), a teoria vem colada à prática, e o percurso todo
mira uma meta concreta — construir uma **base sólida** e chegar preparado ao papel
de arquiteto. É um produto **em produção**: Nuxt 4 (SSR) na Cloudflare, autenticação
e progresso no Supabase.

::doc-overview
::

## 🎯 Para que serve

O RoadMap Academy responde a uma pergunta que quase nenhum curso responde de forma
honesta: **"o que eu preciso estudar, e em que ordem, para sair de júnior e virar
arquiteto?"**. Em vez de uma lista solta de tecnologias, ele entrega um **mapa** —
cada módulo apoia o próximo, cada trilha prepara a seguinte.

::steps
### 1. Trilha Júnior — a base
Fundamentos, terminal, uma linguagem, estruturas de dados, SQL, Git, redes, MVC,
debugging e Docker. **10 módulos.** É onde a base é construída do zero.
👉 [Ver as trilhas e o conteúdo](trilhas-e-conteudo)

### 2. Trilha Pleno / Sênior — profundidade
Bancos avançados, Clean Code, OO, testes, design patterns, performance, mensageria,
microsserviços, cloud, CI/CD, observabilidade, segurança, IA e liderança. **28 módulos.**

### 3. Trilha Arquiteto de Software — o desenho
DDD, arquitetura em camadas, monólito modular, sistemas distribuídos, padrões de
integração, arquitetura evolucionária, documentação e ATAM. **9 módulos.**

### 4. Trilha Arquiteto de Soluções — a visão de negócio
Requisitos não-funcionais, networking e segurança cloud, certificação AWS, System
Design, migração de legado, data engineering, negócios e engenharia de IA. **9 módulos.**
::

## 🧪 Aprender fazendo

A teoria não fica sozinha. A plataforma conecta cada trilha a **9 projetos práticos**
— um por nível de senioridade — que reconstroem sistemas reais (task manager, blog,
e-commerce em microsserviços, e-commerce com DDD, FinanceOS, Vitrine SaaS, migração de
legado, transformação enterprise e uma plataforma de suporte com IA). Cada projeto tem
**código-companheiro** (scaffold de referência em Spring Boot 3 e ASP.NET Core 8).

Além dos projetos, um **playground de System Design** (arrastar-e-soltar) deixa o aluno
desenhar arquiteturas resolvendo exercícios guiados ligados às aulas.

::callout{type="note" title="Status do produto"}
**Em produção / evolução.** As quatro trilhas (53 módulos) e os nove projetos estão
100% escritos como aulas navegáveis, todas em **Java e C#**. Auth e progresso rodam no
Supabase; o deploy é Nuxt SSR na Cloudflare. Ver a **Arquitetura técnica** para o mapa
completo do sistema.
::
